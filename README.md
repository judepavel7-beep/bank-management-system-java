# Bank Management System (Java)

This project is a simple bank account management system developed in Java.

## Features

- Create a bank account
- Deposit money
- Withdraw money
- Store client information in a file
- Read stored client data

## Technologies Used

- Java
- File handling

## Project Structure

- Main.java : program entry point
- Client.java : client information
- Compte.java : bank account operations
- Fichier.java : file management

## How to Run

1. Compile the program

javac *.java

2. Run the program

java Main
